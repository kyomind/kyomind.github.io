我退訂了 GitHub Copilot

2026-06-18 心得

AI 輔助程式設計工具 GitHub Copilot 在今年 6 月 1 號全面改制為 usage-based billing，也就是按用量計費。

作為 2022 年就開始付費的老用戶，我則趕在 5 月 20 號「大限」之前，退訂了原本將在 8 月 16 號到期的 Pro+ 年約，這樣可以拿回一定比例的剩餘退款。

改制到現在已逾兩週，社群反應非常激烈，尤其是個人用戶。

本文分享我決定退訂的判斷與理由，以及我的替代方案。

舊制：大方到不可思議的 PRU

GitHub Copilot 過去採用 PRU（Premium Request Unit）方式計費。

簡單來說，在一次請求之內，不管 AI agent 做了多少步驟，都只消耗一次 Premium Request 額度。這確實非常大方！因此誕生了各式各樣佔便宜的用法XD

最典型的做法就是：先用 0x 模型（不消耗 PRU）撰寫完整的開發計畫，然後只需要用一次或相對少次的 PRU，就可以大量消耗 token 並完成任務。

如此一來，單次請求能跑上十幾二十分鐘——甚至更久！

在這般極限操作之下，你可能只需要付 10 美元（最低等級的月費訂閱），就可以實際使用超過 500 美元等值的 API token。

可想而知，在 AI agent 行為愈來愈複雜的今天，這樣的商業模式肯定無法持續。

尤其在 Subagent（子代理）誕生之後，情況變得更加嚴峻。

在忍無可忍之下，GitHub 決定出手改制了！

新制：Usage-based Billing

現在的 Copilot 改成用 AI Credits 計費，1 Credit 等於 0.01 美元。

所謂 AI Credits，就像賭場裡的籌碼，只是現金的等價替代品，主要是為了方便計算，所以我們直接想成現金就好了。

個人方案分成三種：Pro 月費 10 美元（相當於 1000 Credits，以下同）、Pro+ 月費 39 美元、Max 月費 100 美元。每個方案都含有一定額度的 Credits（含 Flex 額度），用完就要加錢或等下個月。

方案 月費 Base Flex 總額度
Pro $10 $10 $5 ~$15
Pro+ $39 $39 $31 ~$70
Max $100 $100 $100 ~$200

關鍵來了：Credit 的計價方式，跟你自己直接去 OpenAI、Anthropic 官網使用 API 的價格完全一樣。

換言之，真正「額外」給你的，只有那些 Flex 額度！而且這部分竟然是可變動的——GitHub 有權調整 Flex 額度的多寡。

API Token 真的很燒錢

乍看覺得，付 10 元給 15 元額度、39 元給 70 元、100 元給 200 元，似乎還可以呀？

可是瑞凡，API token，真的很燒錢！——消耗速度極快。

要知道，AI 模型推理過程中的那些「自言自語」，可是全都算入 token 消耗的——而且都歸類在更貴的 output token。

以前用 PRU 沒感覺，現在改成按 token 計費，你不禁發現，才一個簡單請求，還沒做多少事，就已經消耗掉幾萬甚至十幾萬的 token。

這些都是錢吶！

別人更大方

Token 燒錢是一回事，但跟對手比，GitHub Copilot 的新方案也顯得太過小氣。

從 X 上的這篇貼文可知，ChatGPT Plus 月費 20 美元，如果週週用滿，實測大約可以達到 700 美元等值的 API token（指 Codex），倍數高達 35 倍，補貼力道驚人。

即使是相對沒那麼慷慨的 Claude Pro，月費 20 美元，最多也能用到 400 美元等值。還是非常大方！

原文中，不同方案的完整測試如下：

方案 月費 最大可能消耗（約）
Anthropic — Claude
claude-pro $20 $400
claude-max-5x $100 $2,000
claude-max-20x $200 $8,000
OpenAI — ChatGPT / Codex
chatgpt-plus $20 $700
chatgpt-pro-5x $100 $3,500
chatgpt-pro-20x $200 $14,000

按照這個表格，GitHub Copilot 即使是 100 美元的 Max 方案，似乎都還沒有 20 美元的競爭對手香。

當然，兩者未必能直接比較，但差距確實相當明顯。

可以理解但無法諒解

老實說，過去的舊模式無法持續，這是完全可以想像的。

所以我基本贊同 GitHub Copilot 應該改成更貼近現實的收費模式，讓服務能盈利且更可持續。

但這次的改動，得不到絕大多數用戶——包括我——的認同。

兩個星期過去，Reddit 和 X.com 上還是罵聲一片，鮮少有人站出來幫 GitHub 聲援。

從超大方到超小氣

畢竟，過去給的福利如此大方，如今急轉直下，難免讓人覺得有幾分「不近人情」。

而且即使社群罵聲連連，官方的態度看起來也十分堅決。

我甚至不禁懷疑，GitHub Copilot 是否像 Perplexity 那樣，試圖改以企業用戶作為主要 TA。而不再那麼在乎個人用戶的感受——畢竟賺的錢有限。

以上純屬個人猜測，反正木已成舟，多說無益。接下來聊聊我的替代方案。

Token 早已過剩😅

退訂之前，我就隱隱發現 token 有點消耗不過來了！

如〈2025 我離不開的 8 項付費訂閱〉一文所言，除了 GitHub Copilot，我同時還訂閱了 ChatGPT Plus、Claude Pro、Google AI Pro 三個 20 美元的方案。

加上我目前正轉職 DevOps，寫程式的量本來就不大。主要是看文件、整理筆記、準備面試，這些任務用現有的訂閱就夠了。

既然需求下降、價格又變貴，退訂是很自然的選擇——雖然我還是會不捨辣！

我更推薦哪個？

如果上述提到的工具你一個都沒有訂閱，而現在考慮要訂閱其中一個試水溫或作為日常的 AI 主力，那我會優先推薦 ChatGPT Plus。

理由有以下 3+1 點。

聊天與 Coding 額度獨立計算

ChatGPT Plus 的額度分成兩種：一般聊天和 Codex。兩者分開計算，不會互相排擠。

這代表你可以一邊用 ChatGPT 處理工作瑣事、查資料、寫文件，同時開 Codex 跑程式任務，不用擔心聊天把 coding 額度吃光。

這點 Claude 就比不上了——Claude 的額度是共享的。

當然，我知道，很多人訂閱它都是衝著 Claude Code 去的，誰跟你聊天！（我個人是聊滿多的☺️）

優秀的 Codex App 使用體驗

Codex 有獨立的桌面 app（macOS、Windows 都有），重點是！它的體驗非常好——設計成熟、互動流暢。大大提升了 Codex 這個服務的能見度！

讓 Google 也忍不住有樣學樣，把他們家的 Antigravity IDE 從 1.0 的 VS Code fork 版本，直接重構成 2.0 的獨立 app 版本，只能說，真香！

而且桌面 app 形式，相較於 CLI，對一般人（非開發者）顯然更加友善且直觀。

更大方的 Token 用量

前面已經算過了，ChatGPT Plus 月費 20 美元，實測最多可以用到約 700 美元等值的 API token，補貼倍數高達 35 倍——這還只是 coding 部分。

這個數字遠超 Claude Pro 的 20 倍，更不用說 Copilot 那感人的 Flex「補貼」了。

額外好處：可以養「龍蝦」

Sam Altman 在 5 月宣布，ChatGPT 訂閱可以直接接入 OpenClaw 使用。

OpenClaw 是一個開源的 AI agent 框架，可以連結各種 LLM，在 WhatsApp、Telegram、Slack 等平台上自動執行任務——管日曆、發信、整理檔案、寫程式都行。

因為它的 logo 是一隻龍蝦，所以社群俗稱「養龍蝦」。

現在 ChatGPT Plus 就能跑，不用另外付錢，算是額外的福利。如此一來，就算你不寫程式，還是可以拿來聊天兼養蝦！

而且據我所知，Claude 和 Google 在這方面都是不鼓勵的，如果把付費訂閱帳號拿來養龍蝦，帳號可能被 ban！

工具只是工具，不要依戀它

很長一段時間裡，我都是 GitHub Copilot 的忠實用戶。

無論是 IDE 競品 Cursor 或 Windsurf，還是如旋風一般席捲開發圈的 Claude Code，都沒有動搖我對它的忠誠。

我始終以 GitHub Copilot 作為我的開發主力。

直到這一次的變動。

無奈退訂之後，我的想法似乎也跟著改變了——工具退回了工具該有的角色。

Claude Code 很好用，Codex 也不錯。哪個順手用哪個，沒有非得選邊站的理由。

通用 vs 專用

我比較不喜歡的是那種 tool-specific（只有特定工具才有）的功能。

比如 Copilot 有個「記憶」功能，會在 repo 底下建一個 memories 資料夾。問題是，這個資料夾只有 GitHub Copilot 會用，其他 agent 根本不認識。

什麼爛東西XD（好啦，prompt files 就是我覺得很成功的設計，我幾乎每個 repo 都有使用）

相較之下，我更偏好通用的機制，像是 AGENTS.md、hooks（現在各家都有了，只是設定略有不同）、MCP、Agent Skills 等。這些東西不管用哪個工具都能用，換工具時不會被綁住。

正因為工具可以互換，所以沒必要為一個定價不合理的服務堅持。

如果 10 美元的 GitHub Copilot Pro 方案能再大方一點，我還是會考慮續訂。

但現在，我才不要💩
